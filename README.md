# Portfolio Website - Muhammad Akmal Abdillah

Portofolio website modern dan responsif dengan nuansa biru dan putih.

## Fitur

- **Desain Responsif**: Tampilan optimal di semua perangkat (desktop, tablet, mobile)
- **Navigasi Smooth**: Scroll halus ke setiap bagian
- **Portfolio Filter**: Filter proyek berdasarkan kategori
- **Animasi Modern**: Efek animasi saat scrolling
- **Form Kontak**: Formulir kontak dengan validasi
- **Back to Top Button**: Tombol untuk kembali ke atas halaman

## Teknologi yang Digunakan

1. **HTML5** - Struktur halaman web
2. **CSS3** - Styling dan layout
3. **JavaScript** - Interaktivitas dan animasi
4. **Bootstrap 5** - Framework CSS untuk responsivitas
5. **Tailwind CSS** - Utility classes untuk styling tambahan
6. **Font Awesome** - Ikon vektor
7. **Google Fonts** - Tipografi
